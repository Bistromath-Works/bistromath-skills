# Artifact templates

Skeletons for the artifacts a session produces. Rules that apply to all of them:
disqualifiers lead (talent/upside never enters until disqualifiers pass), every
disqualifier gets a detection test runnable *before* commitment plus the real
failure that paid for it, and every artifact ends with open items. Fill every
`{…}`; delete rows you can't cite a paid-for lesson for — an uncited disqualifier
is borrowed wisdom, and borrowed wisdom doesn't fire under pressure.

## 1 · Screen (hire / client / partner)

```markdown
# {Role/Relationship} screen — v{n}, {date}

Rule: this screen runs on EVERY candidate, regardless of who referred them.
The more I like the referrer, the more mandatory the screen.

## Disqualifiers (any one fails → no, regardless of talent)

| # | Disqualifier | Detection test (before commitment) | The failure that paid for this row |
|---|---|---|---|
| 1 | {e.g. zero self-attribution in autopsies} | {"walk me through your last three deals that died" — count ownership} | {name/date of the real miss} |
| 2 | | | |

## Only after all disqualifiers pass: upside

- {what great looks like, in observables}

## Open items

- [ ] {reference calls to make — names}
- [ ] {records to pull}
```

## 2 · Kill tripwire (pre-committed exit)

```markdown
# Tripwire — {commitment}, written {date} (before day one)

By day {30/60/90} I must observe: {specific, countable behaviors/results}.
If instead I observe: {the exit condition, specific} → {the exit action}, no
renegotiation with myself.

Reviewer: {who holds me to this — a name, not "me"}
Review date: {calendar date, set now}
Why pre-committed: in the moment, consistency and sunk cost will rationalize.
```

## 3 · Decision log entry

```markdown
### {date} · {decision in one line}

Mechanism found: {the actual thing the session surfaced}
Tendencies named: {incentive-caused bias / liking / authority / sunk cost / …}
Expected outcome: {what should be true, with a number}
Check on: {date} — if wrong, what that falsifies: {…}
```

## 4 · Session homework block

```markdown
## Homework — due {date}

Figures to pull: {which report/export, what number}
Calls to make: {names, one line on what to ask each}
Documents to draft: {which artifact above, for what}
```
